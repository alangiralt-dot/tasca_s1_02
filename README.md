# Tasca S1.02: PHP Bàsics

This repository contains the resolution of the introductory PHP sprint. The goal of this task is to master fundamental programming concepts such as variables, constants, conditional logic, loops, and functions using PHP.

## Objectives
*   Declaration and usage of various data types (integer, double, string, boolean).
*   Operation with variables and constants.
*   Implementation of control structures (if/else, switch, for, while).
*   Code modularization using functions and parameters.
*   Implementation of mathematical algorithms (Sieve of Eratosthenes).

---

## Project Structure

The project is divided into three difficulty levels:

### Level 1: Foundations
*   **Exercise 1:** Variable declaration and type printing + Constants.
*   **Exercise 2:** String manipulation (Uppercase, length, reverse, and concatenation).
*   **Exercise 3:** Arithmetic operations (Integer and Double) and a universal calculator function.
*   **Exercise 4:** Counter function with default parameters.
*   **Exercise 5:** Student grading logic based on percentage thresholds.
*   **Exercise 6:** Probability function (`isBitten()`) using `rand()`.

### Level 2: Practical Logic
*   **Exercise 1:** Phone call cost calculator (Fixed rate for first 3 min + additional charge per minute).
*   **Exercise 2:** Game scoring system (Sum, average, and skill level classification: Beginner, Intermediate, Professional).

### Level 3: Advanced Algorithms
*   **Exercise 1: Sieve of Eratosthenes.** 
    *   Implementation of the classic algorithm to find all prime numbers within a given range (e.g., up to 100).
    *   Optimized using the square root of the limit to reduce iterations.
